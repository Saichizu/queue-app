\# Saichizu's Odyssean Song Queue 🎭



A web-based queue manager built with Streamlit.  

Features:

\- ➕ Join / ➖ Leave Queue  

\- ⏩ Advance Queue  

\- ⏳ Put On Hold (Calypso)  

\- 🏝️ Return from Calypso  

\- 🔀 Change Order  

\- 📋 Copy Output  



\## Run Locally

```bash

pip install streamlit

streamlit run streamlit\_app.py
